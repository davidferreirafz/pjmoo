Atenção esta versão é a ultima antes da liberação oficial do SpaceShooter 0.7
a qual deve ser simultanea para Linux e Windows, prevista para o dia 07 de Setembro de 2005

Agora ja é possivel desfrutar o jogo em 3 fases remodeladas, a primeira fase ganhou um toque
de tutorial, onde durante a fase é passado algumas dicas importantes para todo o jogo.

Foi adicionando um programa chamado "Top10SpaceShooter" o qual sincroniza os recordes locais
com o sistema de recordes globais, e que pode ser visualizado em:
http://pjmoo.codigolivre.org.br/software/top/spaceshooter/galeria.php


Agora com o sistema de instalação serão criados os icons principais do jogo na area de trabalho
e mais alguns itens no menu iniciar >> PJMOO >> SpaceShooter.


Para poder ajudar a validade esta versão do SpaceShooter você pode fazer o donwload em:
http://pjmoo.codigolivre.org.br/software/top/spaceshooter/restrita/SpaceShooter.0.7.bin.win32_DISTRIBUICAO_RESTRITA.exe
Lembro que esta versão não deve ser divulgada, pois não é oficial e apenas é para Windows

Obs.: Antes do lancamento do dia 07 o site ganhará algumas informações adicionais sobre o jogo.

Obs2.: Gostaria de pedir que você publique seu recorde, como sabe agora temos uma galeria online, e a possibilidade de baixar pros jogos locais, para isso basta executar o programa,  "Top10SpaceShooter" ou ir no atalho "SpaceShooter - Atualizar Top Recorde"
